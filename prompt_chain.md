# Prompt Chain Documentation - Task 1 (Future Interns)

## Prompt 1 — Generate YouTube Script
You are a professional YouTube scriptwriter.
Write a highly engaging, storytelling-style YouTube script on the topic: "Manifestation".
Length: 1000 words.
Tone: Friendly + Educative.
Include hook, intro, body with points, and conclusion with CTA.

## Prompt 2 — Thumbnail Ideas
Generate 5 eye-catching YouTube thumbnail ideas for the topic “Manifestation”.
Include:
- Short text (2–3 words)
- Image concept
- Color palette
- Why it attracts viewers.

## Prompt 3 — Scene Breakdown
Break the script into 8–12 scenes suitable for a YouTube video.
For each scene provide:
- Narration
- Visual ideas
- On-screen text
- B-roll keywords.

## Prompt 4 — Thumbnail Image Prompt
Create an AI thumbnail prompt:
"Close-up of a confident young person visualizing success, golden glow behind, purple-gold theme, dramatic lighting, bold text 'Manifest Now', inspirational style, highly clickable YouTube thumbnail."
